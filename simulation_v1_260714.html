Methodology¶

Model¶

Binary choice (Example 1 of main1 with $J=2$) — with a scalar random coefficient $\beta_i \sim F_0$, $\beta_i \perp\!\!\!\perp X_i$:

$$\Pr\left(Y_i = 1 \mid X_i = x\right) \;=\; \int \frac{e^{x\beta}}{1+e^{x\beta}}\, dF_\beta \;=:\; \int g(x, \beta)\, dF_\beta .$$

The choice probability $g(x,\beta)$ is logistic; the observed $Y_i \in \{0,1\}$ is then Bernoulli with that probability. The goal is to estimate $F_0$ nonparametrically. FM's reformulation: write $\beta \overset{d}{=} q_0(U)$ for a base variable $U$ with known law ($U[0,1]$ or $N(0,1)$), estimate the function $q_0$ with a polynomial sieve, then recover $\hat F$ as the distribution of $\hat q(U)$.

DGP¶

Panel with a shared coefficient per individual (FM's 1000 × 8 design; the Meeting Note's numbers "are mostly from FM"):

$$X_{it} \sim N(0, \sigma_X^2), \qquad \beta_i \sim F_0, \qquad Y_{it} \mid X_{it}, \beta_i \sim \text{Bernoulli}\big(g(X_{it}, \beta_i)\big), \qquad t = 1, \dots, T,$$

with $F_0 \in$ {Uniform $[-1,1]$, $N(0,1)$, mixture of two normals} and $R = 50$ Monte-Carlo replications. Setting $T = 1$ (T_PANEL = 1) gives the literal cross-section written in the Meeting Note — but for these three symmetric $F_0$'s that case is unidentified; see To-be-check #1.

Estimation steps¶

Step 1 (base draws). Draw $\{u_m\}_{m=1}^{M}$ i.i.d. from the base law.

Step 2 (simulated sieve MLE). Polynomial sieve $q_a(u) = \sum_{k=0}^{K-1} a_k u^k$, and

$$\hat a \in \arg\max_a \hat L_{n,M}(a), \qquad \hat L_{n,M}(a) = \frac{1}{n} \sum_{i=1}^{n} \log \hat P_i(a), \qquad \hat P_i(a) = \frac{1}{M} \sum_{m=1}^{M} \prod_{t=1}^{T} g\big(X_{it}, q_a(u_m)\big)^{Y_{it}} \big(1 - g(X_{it}, q_a(u_m))\big)^{1-Y_{it}}.$$

At $T=1$ this is the full binary log-likelihood — the one-hot $\sum_j y_{i,j}\log P_j$ form of main1; the Meeting Note writes only the $Y_i$ term as shorthand (To-be-check #2).

Step 3 (CDF reconstruction). With fresh draws $\{v_s\}_{s=1}^{S}$ from the same base law,

$$\hat F(b) = \frac{1}{S} \sum_{s=1}^{S} \mathbb{1}\left\{ \textstyle\sum_{k=0}^{K-1} \hat a_k v_s^k \le b \right\}.$$

1. Functions¶

Step 0 — Model primitives¶

The known link function is the binary logit (Example 1 of main1, $J=2$):

$$g(x, \beta) = \frac{e^{x\beta}}{1 + e^{x\beta}} = \Pr\left(Y_{it} = 1 \mid X_{it} = x,\ \beta_i = \beta\right).$$

Each true distribution $F_0$ is packaged as a TrueDist object exposing draw (used only by the DGP), cdf (used only for evaluation), and its evaluation grid b_grid.

Functions defined below: g_link(x, beta) — logit link; TrueDist — container class; DIST_REGISTRY — the three $F_0$'s from the Meeting Note (uniform, normal, mixture of normals).

# ============================================================
# Step 0 -- Model primitives
# ============================================================

def g_link(x, beta):
    """Logit link  g(x, beta) = e^{x beta} / (1 + e^{x beta}) = P(Y=1 | X=x, beta).

    Example 1 of main1 specialised to J = 2 (binary choice), scalar beta.
    `expit` is the numerically stable logistic CDF. Accepts arrays (broadcasting).
    """
    return expit(x * beta)


class TrueDist:
    """Container for a true distribution F_0 of the random coefficient beta.

    Attributes
    ----------
    name   : label used in tables / figures
    draw   : draw(rng, size) -> beta draws from F_0   (used ONLY inside the DGP)
    cdf    : cdf(b) -> F_0(b), the true CDF            (used ONLY for evaluation)
    b_grid : evaluation grid b for the pointwise / sup-norm comparison
    """
    def __init__(self, name, draw, cdf, b_grid):
        self.name, self.draw, self.cdf, self.b_grid = name, draw, cdf, b_grid


DIST_REGISTRY = {
    # Uniform[-1, 1]
    "uniform": TrueDist(
        name="Uniform[-1,1]",
        draw=lambda rng, size: rng.uniform(-1.0, 1.0, size),
        cdf=lambda b: np.clip((np.asarray(b) + 1.0) / 2.0, 0.0, 1.0),
        b_grid=np.linspace(-1.6, 1.6, 401),
    ),
    # Standard normal N(0,1)
    "normal": TrueDist(
        name="N(0,1)",
        draw=lambda rng, size: rng.standard_normal(size),
        cdf=lambda b: ndtr(np.asarray(b)),
        b_grid=np.linspace(-3.5, 3.5, 401),
    ),
    # Equal-weight mixture of N(-1, 0.5^2) and N(+1, 0.5^2)  (FM's two-normal mixture)
    "mixture": TrueDist(
        name="0.5 N(-1,0.5^2) + 0.5 N(1,0.5^2)",
        draw=lambda rng, size: np.where(rng.random(size) < 0.5, -1.0, 1.0)
                               + 0.5 * rng.standard_normal(size),
        cdf=lambda b: 0.5 * ndtr((np.asarray(b) + 1.0) / 0.5)
                      + 0.5 * ndtr((np.asarray(b) - 1.0) / 0.5),
        b_grid=np.linspace(-3.0, 3.0, 401),
    ),
}

Step 1 — Data-generating process¶

$$X_{it} \sim N(0, \sigma_X^2), \qquad \beta_i \sim F_0, \qquad Y_{it} \mid X_{it}, \beta_i \sim \text{Bernoulli}\big(g(X_{it}, \beta_i)\big), \qquad t = 1,\dots,T,$$

all i.i.d. across $i$, with $\beta_i \perp\!\!\!\perp X_{it}$ and $\beta_i$ shared across the $T$ occasions — the within-person persistence is what identifies the shape of $F_0$ (FM's design).

Function defined below: simulate_data(n, dist, rng) — returns one sample (X, Y) of shape $(n, T)$; the $\beta_i$ stay latent.

# ============================================================
# Step 1 -- Data-generating process (panel with shared beta_i)
# ============================================================

def simulate_data(n, dist, rng):
    """DGP: one sample of n individuals x T_PANEL choice occasions.

    X_it   ~ N(0, SIGMA_X^2)  i.i.d. across (i, t)          (observed)
    beta_i ~ F_0              i.i.d. across i, SHARED across t (LATENT)
    Y_it | X_it, beta_i ~ Bernoulli( g(X_it, beta_i) ),  independent across t given beta_i

    The DGP samples beta_i directly, so it never integrates over F_0: the mixed-logit
    integral P(Y=1|x) = int g(x,b) dF_0(b) only appears at the ESTIMATION stage, where
    beta is unobserved and must be averaged out.  The shared beta_i across the T
    occasions is what carries information about F_0 (FM's 1000 x 8 design); with
    T_PANEL = 1 this collapses to the literal Meeting-Note cross-section (To-be-check #1).

    Returns (X, Y), both float arrays of shape (n, T_PANEL); beta stays latent.
    """
    X = SIGMA_X * rng.standard_normal((n, T_PANEL))
    beta = dist.draw(rng, n)[:, None]                    # (n, 1): shared across t
    Y = (rng.random((n, T_PANEL)) < g_link(X, beta)).astype(float)
    return X, Y

Step 2 — Simulated sieve MLE¶

Sieve space (univariate case, eq. (14) of main1):

$$q_a(u) = \sum_{k=0}^{K-1} a_k u^k =\phi_K(u)^\top a $$

with $a \in \mathbb{R}^K$ and $\phi_K(u)=(1,u,\dots,u^{K-1})^\top$.

The simulated panel objective is

$$\hat L_{n,M}(a) = \frac{1}{n} \sum_{i=1}^{n} \log \hat P_i(a), \qquad \hat P_i(a) = \frac{1}{M} \sum_{m=1}^{M} \underbrace{\prod_{t=1}^{T} g_{imt}^{Y_{it}} (1 - g_{imt})^{1-Y_{it}}}_{=:\ L_{im}}, \qquad g_{imt} = g\big(X_{it},\, q_a(u_m)\big),$$

maximized by BFGS with the analytic gradient. Using the logit score $\frac{\partial}{\partial \beta} \log\left[g^y (1-g)^{1-y}\right] = (y - g)\,x$ and the chain rule $\partial q_a(u_m) / \partial a_k = u_m^k$:

$$\frac{\partial \hat P_i}{\partial a_k} = \frac{1}{M} \sum_{m=1}^{M} L_{im}\, S_{im}\, u_m^k, \qquad S_{im} = \sum_{t=1}^{T} \big(Y_{it} - g_{imt}\big) X_{it}, \qquad \frac{\partial \hat L}{\partial a_k} = \frac{1}{n} \sum_{i=1}^{n} \frac{1}{\hat P_i} \frac{\partial \hat P_i}{\partial a_k}.$$

Numerical stability: $y \log g + (1-y)\log(1-g) = y z - \log(1 + e^{z})$ with $z = x\, q_a(u_m)$, computed as y*z - logaddexp(0, z) — this avoids overflow of $e^z$ when the index is large (which it can be after summing over the $T$ occasions).

Functions defined below: draw_base — draws $u_m$; build_design — monomial matrix $[u_m^k]$; neg_loglik_and_grad — $(-\hat L_{n,M}, -\nabla_a \hat L_{n,M})$; canonical_start — deterministic initial $a$; fit_sieve_mle — BFGS wrapper returning $\hat a$ for one $K$.

# ============================================================
# Step 2 -- Simulated sieve MLE
# ============================================================

def draw_base(base_kind, size, rng):
    """Draw base random variables:  u ~ U[0,1] ('uniform') or u ~ N(0,1) ('normal')."""
    if base_kind == "uniform":
        return rng.random(size)
    if base_kind == "normal":
        return rng.standard_normal(size)
    raise ValueError(f"unknown base_kind: {base_kind}")


def build_design(u, K):
    """Monomial design matrix [u_m^k] of shape (len(u), K), columns k = 0,...,K-1.

    Row m is (1, u_m, u_m^2, ..., u_m^{K-1}), so the sieve polynomial is
    q_a(u_m) = (design @ a)_m = sum_k a_k u_m^k.
    """
    return np.vander(u, K, increasing=True)


def neg_loglik_and_grad(a, X, Y, U_design):
    """Negative simulated sieve log-likelihood -L_hat_{n,M}(a) and its analytic gradient.

    Panel likelihood (individual i's T choices share one beta):

      L_hat_{n,M}(a) = (1/n) sum_i log P_hat_i(a),
      P_hat_i(a)     = (1/M) sum_m  prod_t  g_imt^{Y_it} (1-g_imt)^{1-Y_it},
      g_imt          = g( X_it, q_a(u_m) ).

    Numerically stable log:  y log g + (1-y) log(1-g) = y z - log(1 + e^z),  z = x q_a(u),
    computed as  y*z - logaddexp(0, z).

    Analytic gradient (logit score d/d(beta) log[g^y (1-g)^{1-y}] = (y-g) x):
      dP_hat_i/da_k = (1/M) sum_m  L_im  S_im  u_m^k,
      S_im = sum_t (Y_it - g_imt) X_it,   L_im = prod_t g^y (1-g)^{1-y}.

    Returns (-L_hat, -grad) for scipy.optimize.minimize(jac=True).
    """
    n, T = X.shape
    M = U_design.shape[0]
    beta_m = U_design @ a                            # (M,)   q_a(u_m)
    logL = np.zeros((n, M))                          # log prod_t g^y (1-g)^{1-y}
    S = np.zeros((n, M))                             # sum_t (y - g) x   (score factor)
    for t in range(T):
        z = np.outer(X[:, t], beta_m)                # (n,M)  X_it * q_a(u_m)
        y_t = Y[:, t][:, None]
        logL += y_t * z - np.logaddexp(0.0, z)       # y log g + (1-y) log(1-g)
        S += (y_t - expit(z)) * X[:, t][:, None]
    L = np.exp(logL)                                 # (n,M)  individual-by-draw likelihood
    P = np.clip(L.mean(axis=1), 1e-300, None)        # (n,)   P_hat_i(a)
    ll = np.mean(np.log(P))
    grad = ((L * S / P[:, None]) @ U_design).mean(axis=0) / M
    return -ll, -grad


def canonical_start(base_kind, K):
    """Deterministic starting value for the sieve coefficients a.

    uniform base : q(u) = -2 + 4u   maps U[0,1] onto [-2, 2]
    normal  base : q(u) = u         i.e. start from beta ~ N(0,1)
    Higher-order coefficients start at 0.
    """
    a0 = np.zeros(K)
    if base_kind == "uniform":
        a0[0], a0[1] = -2.0, 4.0
    else:
        a0[1] = 1.0
    return a0


def fit_sieve_mle(X, Y, base_kind, K, rng):
    """Step 2: maximize L_hat_{n,M} over a in R^K by BFGS, for one sieve order K.

    Returns a_hat: the best coefficient vector.
    """
    u = draw_base(base_kind, M_SIM, rng)
    U_design = build_design(u, K)
    a0 = canonical_start(base_kind, K)
    res = minimize(neg_loglik_and_grad, a0, args=(X, Y, U_design),
                   jac=True, method="BFGS", options={"maxiter": 500})
    return res.x

Step 3 — Rebuild $\hat F$ and the sup-distance¶

With i.i.d. base draws $\{v_s\}_{s=1}^{S}$,

$$\hat F(b) = \frac{1}{S} \sum_{s=1}^{S} \mathbb{1}\big\{ q_{\hat a}(v_s) \le b \big\},$$

a right-continuous empirical CDF (i.e. the operator $H\{\hat q\}$ in equation (19) of main1) evaluated by simulation. The comparison metric for Experiment C is $\max_b |\hat F(b) - F_0(b)|$ over b_grid.

Functions defined below: Fhat_eval(a_hat, base_kind, K, b_grid, rng) — empirical CDF of $q_{\hat a}(v_s)$; sup_distance(F_hat, F_true) — sup-norm over the grid; run_once(...) — one full Steps 1–3 pass, reused by all three experiments.

# ============================================================
# Step 3 -- Rebuild F-hat and the sup-distance
# ============================================================

def Fhat_eval(a_hat, base_kind, K, b_grid, rng):
    """Step 3: estimator of the CDF of beta, evaluated on b_grid.

    F_hat(b) = (1/S) sum_s 1{ q_a_hat(v_s) <= b },  {v_s} fresh i.i.d. base draws.

    Implementation: sort q(v_s) once, then F_hat(b) = #(q <= b)/S via
    searchsorted (right-continuous empirical CDF).
    """
    v = draw_base(base_kind, S_CDF, rng)
    q_v = np.sort(build_design(v, K) @ a_hat)
    return np.searchsorted(q_v, b_grid, side="right") / q_v.size


def sup_distance(F_hat, F_true):
    """max_b |F_hat(b) - F_0(b)| over the evaluation grid (approximates the sup-norm)."""
    return float(np.max(np.abs(F_hat - F_true)))


def run_once(n, dist, base_kind, K, rep_seed):
    """One full pass of Steps 1-3 on one simulated dataset.

    Returns (F_hat on dist.b_grid, sup-distance to the true CDF).
    Reused by Experiments A, B and C.
    """
    rng = np.random.default_rng(rep_seed)
    X, Y = simulate_data(n, dist, rng)
    a_hat = fit_sieve_mle(X, Y, base_kind, K, rng)
    F_hat = Fhat_eval(a_hat, base_kind, K, dist.b_grid, rng)
    return F_hat, sup_distance(F_hat, dist.cdf(dist.b_grid))

Experiment B — pointwise Monte-Carlo comparison ($\hat F$ vs $F_0$)¶

The Meeting-Note comparison "$\hat F(b)$ v.s. $F_\beta(b)$ point-wise", in the style of FM Figure 1: for each true $F_0$ (rows) and each base law (columns), $R = 50$ replications of the DGP at $n = 1000$; plot the true CDF, the pointwise mean of $\hat F$, and the pointwise 5–95% envelope across replications.

Reading the output: a tight envelope hugging the black curve = the estimator recovers that shape well; envelopes widening at the tails or smoothing out the mixture's plateau reveal where the polynomial sieve struggles at this $K$.

Experiment C — is $\max_b |\hat F - F_0|$ convergent, and how fast?¶

For every $F_0$, each base law, and $K \in \{3,4\}$, compute the mean sup-distance over $R = 50$ replications at each $n \in$ N_GRID, then fit a line in log-log space:

$$\log \mathbb{E}\Big[\max_b |\hat F - F_0|\Big] \approx c + \gamma \log n .$$

The fitted slope $\gamma$ is the empirical convergence-rate exponent ($\gamma = -0.5$ would be the parametric $n^{-1/2}$ rate).

Function defined below: mc_supdist(dist, base_kind, K, n, R, tag) — mean and sd of the sup-distance at one $(n, \text{base}, K)$.

def mc_supdist(dist, base_kind, K, n, R, tag):
    """Mean and sd of the sup-distance over R replications at sample size n."""
    children = np.random.SeedSequence([SEED, 200, tag]).spawn(R)
    sds = np.empty(R)
    for r in range(R):
        _, sds[r] = run_once(n, dist, base_kind, K, rep_seed=children[r])
    return sds.mean(), sds.std(ddof=1)

To be check¶

1. Cross-section (Meeting Note, literal) vs panel (FM) — the DGP as written is uninformative for these $F_0$'s.

The Meeting-Note DGP line is a cross-section: one binary $Y_i$ per individual. For the three $F_0$'s the note names (uniform, normal, mixture of normals — all symmetric about 0) the cross-section carries exactly zero information about $F_0$: since $\frac{e^{z}}{1+e^{z}} + \frac{e^{-z}}{1+e^{-z}} = 1$, any $F_0$ symmetric about 0 gives $\Pr(Y_i=1 \mid X_i = x) = \int g(x\beta)\,dF_0 \equiv \tfrac12$ for every $x$ — every symmetric distribution is observationally equivalent. Diagnostics during development confirm it empirically: mean sup-distance flat at ≈ 0.25–0.35 from $n = 1000$ to $16{,}000$, with the fitted likelihood exceeding the likelihood at the true $q_0$ (wandering on a flat objective). FM's actual simulations use a panel — 1000 individuals × 8 choices with $\beta_i$ shared within individual — whose within-person correlation is what identifies $F_0$; the Meeting Note says its numbers "are mostly from FM".

Current solution: panel DGP with T_PANEL = 8 (FM's design) as the default; the estimator then reaches sup-distances ≈ 0.03–0.10 at $n = 1000$.

Alternatives: (i) T_PANEL = 1 with an asymmetric $F_0$ (e.g. $N(1,1)$, or FM's shifted lognormal) — the cross-section is then informative, though still weakly; (ii) T_PANEL = 1 with the symmetric $F_0$'s to demonstrate the non-identification.

2. The likelihood in the Meeting Note has only the $Y_i$ term. The note writes $\hat L_{n,M}(a) = \frac{1}{n}\sum_i Y_i \log(\cdot)$. Taken literally, the argmax pushes $\hat P_i \to 1$ for every $i$ (the $Y_i=0$ individuals never enter the objective, so nothing penalizes $\hat P_i \to 1$; degenerate).

Current solution: the full likelihood — for $T=1$, $Y_i \log \hat P_i + (1-Y_i)\log(1-\hat P_i)$, which is the one-hot $\sum_j y_{i,j} \log P_j$ objective of main1 at $J = 2$; for the panel, $\log$ of the product over occasions. The note is shorthand.

3. Distribution of $X$ is never specified.

FM use $x \sim N(0,1)$ (standard normal) together with a scale parameter $\sigma = 2$ inside the logit, $P(1\mid\beta,x) = 1/(1+e^{\sigma(\beta-x)})$ — there $\sigma$ controls how steep the choice probability is in the index, and $\beta$ enters as a random location.

Our model $g(x,\beta) = \Lambda(x\beta)$ has $\beta$ as a random slope and no separate $\sigma$.

Current solution: fold a comparable steepness into the regressor, $X_{it} \sim N(0, 2^2)$; larger $|x|$ makes $g(x,\beta)$ vary sharply in $\beta$, which is what identifies the tails of $F_0$.

Alternatives: $X \sim N(0,1)$ (slower tail recovery), or add an explicit scale parameter to match FM's parameterization exactly.

4. Bounded support assumption vs normal DGPs. Assumption 1.1 of main1 requires $\text{supp}(F_0) \subseteq [-M, M]$; the $N(0,1)$ and mixture DGPs violate this literally.

Current solution: keep them untruncated (FM does the same).

Alternative: truncate at $\pm 4$ sd — indistinguishable in practice.

5. Method of optimization when computing the argmax.

The estimator needs $\hat a = \arg\max_a \hat L_{n,M}(a)$, but the Meeting Note and main1 do not prescribe how to solve it.

Current solution: unconstrained BFGS (scipy.optimize.minimize(method="BFGS")) with an analytic gradient (jac=True), maxiter=500, a single deterministic start per $K$ (canonical_start). Reason: the objective is smooth and low-dimensional ($K \le 4$), so a quasi-Newton method reaches superlinear convergence using only gradients — no Hessian, no numerical differentiation.
